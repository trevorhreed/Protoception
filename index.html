Introduction.

Protoception is a simple Node.js application for hosting an SPA prototype. Additionally, it provides a JSON database accessible at localhost:3000/data/. You can GET, PUT, POST, and DELETE any JSON data under this location. You can even do some querying using the SpahQL API when doing a GET.
